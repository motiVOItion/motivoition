10 After Effects Tips to Speed Up Your Workflow

Professional techniques to work faster and smarter in Adobe After Effects

Jamie Chen
July 22, 2023
12 min read
Tutorial, Software

Adobe After Effects is a powerful tool for motion graphics and visual effects, but its complexity can sometimes slow down even experienced users. Over my decade of working with After Effects, I've discovered numerous tricks and techniques that significantly speed up my workflow. In this article, I'll share my top 10 tips that will help you work faster and more efficiently.

1. Master Essential Keyboard Shortcuts

1

Customize and Memorize Key Shortcuts

Keyboard shortcuts are the single biggest time-saver in After Effects. While everyone knows basics like Spacebar for playback, true efficiency comes from customizing your own shortcuts and using lesser-known combinations.

Ctrl/Cmd + Alt/Opt + Shift + C

Create a new composition from selected layers

Ctrl/Cmd + Alt/Opt + T

Show/hide all layer properties with keyframes

U

Show all animated properties for selected layers

Ctrl/Cmd + Shift + D

Split layer at current time indicator

Pro Tip: Go to Edit > Keyboard Shortcuts to customize your own. I recommend creating shortcuts for frequently used effects like Gaussian Blur, Fill, and Transform.

2. Utilize Adjustment Layers for Global Changes

2

Work Non-Destructively with Adjustment Layers

Instead of applying effects directly to multiple layers, use Adjustment Layers (Layer > New > Adjustment Layer) to apply effects globally. This maintains your original layers intact and allows for quick changes.

Without Adjustment Layer
  • Apply effect to each layer individually
  • Hard to make uniform changes
  • Inconsistent results across layers
  • Time-consuming to adjust
With Adjustment Layer
  • Apply effect once to adjustment layer
  • Affects all layers beneath it
  • Easy to make global changes
  • Consistent results across project

This technique is perfect for color grading, adding film grain, or applying global blurs and glows.

3. Create and Use Master Properties

3

Streamline Repetitive Elements

If you're using the same graphic elements multiple times (like logos, lower thirds, or icons), convert them to Master Properties. This allows you to make changes once and have them update across all instances.

// How to create Master Properties:
1. Select your pre-composed layer
2. Right-click and choose "Create Master Properties"
3. Drag the master comp from Project panel to create instances
4. Edit the master to update all instances simultaneously

This is especially useful for client work where logos or brand elements might change during the revision process.

4. Optimize Preview and Rendering

4

Speed Up Your Playback

Slow previews can kill your momentum. Here's how to optimize:

  • Use Quarter Resolution: Work at half or quarter resolution (click on the resolution dropdown next to the composition window).
  • Enable Caps Lock: This prevents After Effects from updating the composition window while you work.
  • Purge Memory: Regularly go to Edit > Purge > All Memory & Disk Cache to free up RAM.
  • Render & Replace: For complex precomps, right-click and choose "Render and Replace" to convert to a video file temporarily.

5. Harness the Power of Expressions

5

Automate Repetitive Animations

Expressions can automate animations and create dynamic relationships between properties. You don't need to be a programmer to use basic expressions.

// Basic expression to make rotation follow position:
wiggle(2, 30) // Creates organic wiggle animation

// Link opacity to audio amplitude:
temp = thisComp.layer("Audio").effect("Both Channels")("Output")[0];
linear(temp, 0, 0.5, 0, 100)

// Loop animation seamlessly:
loopOut("cycle", 0)

Alt/Option + click on the stopwatch next to any property to add an expression. Start with simple ones and build up your library.

6. Organize with Color Labels and Folders

6

Maintain a Clean Workspace

A cluttered timeline is a productivity killer. Use these organization techniques:

  • Color Labels: Right-click on layers to assign colors (e.g., blue for text, green for graphics, red for effects).
  • Layer Markers: Use markers (press * on numeric keypad) to note important timing points.
  • Pre-compose: Group related layers into pre-comps to simplify your main timeline.
  • Search: Use the search bar in the timeline to quickly find layers by name or effect.

7. Create Custom Animation Presets

7

Build Your Animation Library

If you find yourself recreating the same animations (fades, slides, bounces), save them as Animation Presets.

  1. Animate your layer with the desired properties
  2. Select all the keyframes in the timeline
  3. Go to Animation > Save Animation Preset
  4. Name your preset and save it
  5. Access it later from the Effects & Presets panel

You can even share these presets between computers or with team members.

8. Utilize the Graph Editor Like a Pro

8

Create Smooth, Professional Animations

The Graph Editor is intimidating but essential for professional motion. Instead of just using Easy Ease (F9), learn to manipulate bezier handles directly:

  • Separate Dimensions: Right-click on Position property and choose "Separate Dimensions" to animate X and Y independently.
  • Hold Keyframes: Convert to hold keyframes (Ctrl/Cmd + Alt/Opt + H) for sudden changes.
  • Value Graph vs Speed Graph: Toggle between them to control either the property value or its rate of change.
  • Easy Ease Custom: After applying Easy Ease, adjust the influence handles for perfect timing.

9. Scripting for Bulk Operations

9

Automate Repetitive Tasks

For tasks you do repeatedly (renaming layers, adjusting effects, creating templates), consider learning basic scripting or using existing scripts.

Essential After Effects Scripts:

  • Motion v2 - Streamlined animation workflow
  • Overlord - Bridge between Illustrator and AE
  • EaseCopy - Copy and paste easing between keyframes
  • True Comp Duplicator - Duplicate comps with all dependencies
  • Render Garden - Distributed rendering
  • Flow - Create and save custom easing curves

Many of these scripts are free or low-cost and can save hours of work.

10. Establish a Consistent Project Structure

10

Create Templates for Repeated Projects

If you work on similar projects regularly (social media videos, explainers, corporate presentations), create template projects with:

  • Pre-set compositions at common resolutions (1920x1080, 1080x1920, etc.)
  • Organized folders for assets, audio, graphics, and renders
  • Standard adjustment layers for color correction
  • Common lower thirds, titles, and transition presets
  • Render settings templates for different outputs

Save these as .aep files in a "Templates" folder for quick access on new projects.

Bonus: Essential Plugins to Consider

While native After Effects is powerful, these plugins can dramatically improve specific workflows:

  • Element 3D: For 3D objects and text without leaving AE
  • Trapcode Suite: Particular and Form for particle systems
  • Duik: Free rigging tool for character animation
  • Boris FX Continuum: Comprehensive effects package
  • RE:Vision Effects: Twixtor for time remapping and more

Conclusion

Mastering After Effects is a journey, and efficiency comes with practice and intentional workflow development. Start by implementing 2-3 of these tips in your next project, and gradually incorporate more as they become second nature.
